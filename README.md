## Running as DEV
#### Installing Virtualenv:
- `sudo pip3 install virtualenv`
- `mkdir venv`
- `which python3`
- `virtualenv --python='/path/to/python3' venv`
- `source venv/bin/activate`
- *To stop the virtual environment run:* `deactivate`
- *Top remove all dependencies run:* `rm -r venv`
